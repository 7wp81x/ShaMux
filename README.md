## Termux Installation

```
$ apt update && apt upgrade -y
$ apt install git -y
$ cd ~/
$ git clone https://github.com/Mrp1r4t3/ShaMux
$ cd ShaMux/
$ chmod +x install.sh
$ chmod +x shamux.sh
$ ./install.sh
$ ./shamux.sh
```
